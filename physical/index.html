Physcial security

This blog post is about physical security.

Physical security is a broad topic that describes security measures implemented to prevent unauthorized access to facilities, equipment, or any number of internal resources, as well as provide protection to equipment and personnel. From the locks on the doors, to the security guards, to the fire prevention systems, to permieter intrusion detection, physical security is a denial or protection mechanism that restricts access to secured areas or detects unsafe conditions. Effective application of physical security principles relies on defense in depth, or multiple layers of independent systems.

Physical security systems prevent people and equipment from accidental damage, theft, vandolism, sabotage, espionage, terrorism, and criminal activity. This can include passive systems that ensure the ability to audit access, like CCTV (Closed Circuit TeleVision) cameras that record video. Physical security cna also include active systems that react to unauthorized access, like alarms or security guards. Properly implemented security measures should deter potential intruders, detect intrusions and monitor / record the incident, and trigger appropriate responses.

Access control systems also fall under the umbrella of physical security. Access control can be something as simple as a lock and key on a door, or as complex as biometric scanners or RFID badges. When entry points to secure areas need stricter controls than are provided by a doorway, a mechanical security device called a mantrap, also known as an airlock or security portal, can be used. A mantrap is a second set of security doors that only operate after the first set closes, creating a secure container to restrict the number of people that can pass through at a time. The mantrap can be manual, where a security guard manually opens a second door after the first one is closed, or automatic, where an automated system opens the second door after the first one closes.

Electronic Security Systems (or ESS) are physical security systems that integrate into a facilities Access Control Systems. This can inclued CCTV video sureveilance, physical access control systems, or intrusion detection systems. PACS components include, but are not limited to, card readers, door controllers, magnetic locks, turnstiles, request to exit (or REX) devices, door position sensors, electric door strikes, control hardware, and the physical network infrastructure that provides connectivity to the various components.

Some security systems, like security lighting, fences, or signage indicating the area is under surveilance, can serve as a deterent to malicious actors. By inidicating to attackers that security measures are in place, they may be convinced that successful penetration is unlikely due to strong defenses and a more vulnerable target should sought. Architecural design can also be used to create area of high visiility, therby using natural surveillance to increase the visibility of unauthorized activity. By incorporating different security elements as part of a carefully planned design, stronger overall security can be acheived.

Physical security can also include cable locks that prevent equipment, like monitor or computers, from being stolen off or desks.

Security assessments need to be done regularly to ensure the efficacy of existing systems.
